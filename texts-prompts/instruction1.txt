**Getting started:**
```bash
unzip family-tree-nextjs.zip
cd family-tree
npm install
node scripts/generate-placeholders.js   # creates temp SVG images
npm run dev                              # → localhost:3000
```

**Adding your images** — drop them into `/public/assets/` with these names:
```
familyNo1father.jpg    familyNo1mother.jpg
familyNo2father.jpg    familyNo2mother.jpg
familyNo3father.jpg    familyNo3mother.jpg
familyNo4father.jpg    (no mother for Jon Snow)
```

**To export to YouTube MP4:**
```bash
npm install puppeteer
# install ffmpeg on your system
node scripts/export-frames.js
# → output/family-tree-video.mp4 (1920×1080, H.264, 20Mbps, YouTube-ready)
```

**To change the lineage** — edit only `lib/lineageData.ts`. Everything else is automatic.
