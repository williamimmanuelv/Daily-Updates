## 🛠 Step-by-Step: Run the Project

### Step 1 — Install Node.js (if not already)
Go to **https://nodejs.org** and download the **LTS version**. Install it. Verify:
```bash
node --version   # should say v18 or higher
npm --version
```

### Step 2 — Extract and install
```bash
unzip family-tree-nextjs.zip
cd family-tree
npm install
```

### Step 3 — Generate placeholder images
```bash
node scripts/generate-placeholders.js
```
This creates temporary silhouette images in `/public/assets/`. Replace them with your real images any time (same filenames).

### Step 4 — Run the website
```bash
npm run dev
```
Open **http://localhost:3000** in your browser. You'll see the intro screen → click **BEGIN THE LINEAGE**.

---

## 🎬 Step-by-Step: Export to YouTube Video

### Step 5 — Install FFmpeg
- **Mac:** `brew install ffmpeg`
- **Windows:** Download from https://ffmpeg.org/download.html → add to PATH
- **Ubuntu:** `sudo apt install ffmpeg`

Verify: `ffmpeg -version`

### Step 6 — Install Puppeteer
```bash
npm install puppeteer
```
This downloads a headless Chrome browser (~170MB, one-time).

### Step 7 — Make sure the dev server is running
```bash
npm run dev       # keep this terminal open
```

### Step 8 — Export frames + render video (new terminal)
```bash
node scripts/export-frames.js
```
This will: capture ~300–400 PNG frames → pipe them to FFmpeg → output `output/family-tree-video.mp4`

### Step 9 — Upload to YouTube
The output file is already YouTube-optimised: **H.264, 1920×1080, 30fps, 20Mbps, faststart**.
